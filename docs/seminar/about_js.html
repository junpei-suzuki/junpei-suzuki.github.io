担当教員について

更新日

2026年4月2日

実証社会科学演習担当教員の鈴木淳平（すずき・じゅんぺい）と申します。

私の公式の経歴や研究についてはCVやらresearchmapに書いてありますから、繰り返してもあまり面白くないと思います。むしろここではもう少し個人的な話をしたいと思います。

経歴

出身は神奈川県東京都町田市というところで、基本的に東京圏内で生まれ育った人間です。小学校と中学校は地元の公立に通いました。高校は東京都立の戸山高校というところを出ました。

高校を出た後、東京大学の受験に失敗して早稲田大学政治経済学部政治学科に進学し、そのまま大学院（修士・博士課程）まで行きました。2012年に早稲田に入学し、博士を取得したのが2024年ですから、合計12年間早稲田に通ったことになります。常人の3倍時間を大学に費やしましたので、一年二年留年したくらいの話では動じません。

大学学部の時は飲み会や麻雀ばかりをしていてあまり品行方正な方ではなかったと思います。そんな私がなぜ大学院に進学したのかというと、もともと政治に関心があって、大学の授業自体は結構面白かったからです。成績は悪くなかったですし、4年間で単位は一回も落としたことはありません。

しかし進学した理由としては、いわゆる「就職活動」をしたくなかったというのも大きいです。学部の時分はミシェル・フーコーなどにかぶれてだいぶ斜に構えていて、「資本主義経済に規律されたくない」などと息巻いていました。ただ教育者になった今では、このような姿はあるべき姿ではないと思っています。皆さんはちゃんと就活を頑張りましょう

大学院で博士課程を終えた後は、東京大学社会科学研究所（社研）というところで2年間ポスドクをしました。ご自分で調べてもらった方が早いとは思いますが、ポスドクというのは博士取得後の若手研究者の修行期間のようなものです。大学などの学術界は普通の民間企業とは異なり、最初から終身雇用の仕事につけることはまずありません。最初は期間限定で色々な研究機関を渡り歩き、終身雇用の職を探すというプロセスを経ます。私も最大3年の予定で社研に在籍しておりました。社研での思い出話はこのpodcastでしているので、興味があれば聞いてみてください。

社研の居心地は結構良く、丸々3年間でいるつもりであったのですが、しかし人生というのはよくわかりません。運よく終身雇用の職が決まり、この4月に駒澤大学に着任することになりました。正直駒澤とはこれまで縁もゆかりも薄く、事情をあまり理解しているわけではありません。なので学生の皆さんにも色々教えていただきたいと思っています。特に美味しい店の情報は大歓迎です。
